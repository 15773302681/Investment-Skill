## 概览
本框架提供财务指标分析的系统方法，帮助评估公司的盈利能力、成长性、财务风险和投资价值，为投资决策提供量化依据。

## 盈利能力指标

### 核心指标

#### 1. 毛利率（Gross Margin）
```
毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%
```
- **含义**：反映产品定价能力和成本控制能力
- **分析要点**：
  - 纵向对比：观察毛利率变化趋势
  - 横向对比：与同行业竞争对手对比
  - 拆解分析：产品结构变化、原材料价格、提价/降价策略
- **标准参考**：
  - 消费品：30%-60%
  - 科技硬件：20%-40%
  - 软件服务：50%-80%

#### 2. 净利率（Net Margin）
```
净利率 = 净利润 / 营业收入 × 100%
```
- **含义**：反映最终盈利能力
- **分析要点**：
  - 毛利率-净利率的差值反映费用管控能力
  - 观察非经常性损益的影响
- **标准参考**：
  - 优秀企业：>15%
  - 一般企业：5%-15%
  - 低毛利行业：<5%

#### 3. ROE（净资产收益率）
```
ROE = 净利润 / 平均净资产 × 100%
```
- **含义**：股东投入资本的回报率，巴菲特最看重的指标
- **分析要点**：
  - 杜邦拆解：ROE = 净利率 × 资产周转率 × 权益乘数
  - 高ROE可持续性：区分高利润驱动 vs 高杠杆驱动
- **标准参考**：
  - 优秀：>20%
  - 良好：15%-20%
  - 一般：10%-15%

#### 4. ROA（总资产收益率）
```
ROA = 净利润 / 平均总资产 × 100%
```
- **含义**：反映资产使用效率，剔除杠杆影响
- **分析要点**：
  - ROA与ROE的差值反映杠杆水平
  - 重资产行业ROA通常较低
- **标准参考**：
  - 轻资产行业：>10%
  - 重资产行业：3%-8%

### 盈利质量评估
- **扣非净利润占比**：非经常性损益占比越高，盈利质量越差
- **经营性现金流/净利润**：>1为佳，<0.8需警惕
- **应收账款占比**：应收账款增长快于收入增长，需关注回款风险

## 成长性指标

### 核心指标

#### 1. 营收增速
```
营收增速 = (本期营收 - 上期营收) / 上期营收 × 100%
```
- **分析维度**：
  - 单季度增速 vs 同比增速
  - 三年复合增长率（CAGR）
  - 增速趋势：加速/减速/稳定
- **标准参考**：
  - 高成长：>30%
  - 稳健成长：15%-30%
  - 成熟期：0%-15%

#### 2. 净利润增速
```
净利润增速 = (本期净利润 - 上期净利润) / 上期净利润 × 100%
```
- **分析要点**：
  - 利润增速应与营收增速匹配
  - 利润增速 > 营收增速 → 规模效应显现
  - 利润增速 < 营收增速 → 价格战或费用上升

#### 3. 业务结构变化
- 新业务收入占比提升
- 高毛利产品占比变化
- 区域市场扩张情况

### 可持续性评估
- **内生增长 vs 外延增长**：并购带来的增长需评估整合效果
- **市场份额变化**：营收增速 vs 行业平均增速
- **订单/合同负债**：前瞻性指标，反映未来收入确定性

## 财务健康度指标

### 偿债能力

#### 1. 资产负债率
```
资产负债率 = 总负债 / 总资产 × 100%
```
- **分析要点**：
  - 过高杠杆增加财务风险
  - 需结合行业特性（金融、地产可接受较高杠杆）
- **标准参考**：
  - 风险较高：>70%
  - 适中：40%-70%
  - 保守：<40%

#### 2. 流动比率
```
流动比率 = 流动资产 / 流动负债
```
- **标准参考**：
  - 理想值：>2
  - 警戒线：<1.5

#### 3. 速动比率
```
速动比率 = (流动资产 - 存货) / 流动负债
```
- **含义**：剔除存货后的短期偿债能力
- **标准参考**：
  - 理想值：>1

### 现金流健康度
- **经营性现金流净额**：持续为正且增长为佳
- **自由现金流**：经营现金流 - 资本开支，反映真实现金创造能力
- **现金周期**：
  ```
  现金周期 = 存货周转天数 + 应收账款周转天数 - 应付账款周转天数
  ```
  - 越短越好，负数为佳（占用供应商资金）

## 运营效率指标

#### 1. 存货周转天数
```
存货周转天数 = 365 / (营业成本 / 平均存货)
```
- **含义**：存货销售速度，反映库存管理效率
- **标准参考**：
  - 快消品：30-60天
  - 电子产品：45-90天
  - 耐用品：90-180天

#### 2. 应收账款周转天数
```
应收账款周转天数 = 365 / (营业收入 / 平均应收账款)
```
- **含义**：回款速度，反映议价能力和客户质量
- **标准参考**：
  - 优秀：<30天
  - 良好：30-60天
  - 关注：>90天

#### 3. 总资产周转率
```
总资产周转率 = 营业收入 / 平均总资产
```
- **含义**：资产使用效率
- **标准参考**：
  - 轻资产行业：>1.5
  - 重资产行业：0.5-1.0

## 估值指标

### 绝对估值
- **DCF（现金流折现法）**：
  - 适用于现金流稳定的成熟企业
  - 核心参数：自由现金流、折现率、永续增长率

### 相对估值

#### 1. PE（市盈率）
```
PE = 股价 / 每股收益
PE = 总市值 / 净利润
```
- **适用场景**：盈利稳定的公司
- **分析要点**：
  - 历史PE分位数（当前PE在过去5年/10年的位置）
  - 相对行业PE倍数
  - PE = PEG × 增速，PEG=1为合理估值
- **注意**：亏损公司PE失效

#### 2. PB（市净率）
```
PB = 股价 / 每股净资产
PB = 总市值 / 净资产
```
- **适用场景**：重资产行业、金融、周期股
- **分析要点**：
  - ROE是关键驱动因素：PB ≈ ROE × 10
  - ROE=20%的企业，合理PB≈2倍
- **注意**：轻资产行业PB参考价值低

#### 3. PS（市销率）
```
PS = 股价 / 每股营收
PS = 总市值 / 营业收入
```
- **适用场景**：亏损公司、高成长公司
- **分析要点**：
  - 需结合净利率预期：PS = PE × 净利率
  - 高成长公司可接受高PS

#### 4. EV/EBITDA
```
EV/EBITDA = (市值 + 净债务) / EBITDA
```
- **适用场景**：资本密集型行业、跨国比较（排除税率和折旧差异）

### 估值判断框架
- **低估**：历史PE分位数<30%或PEG<0.8
- **合理**：历史PE分位数30%-70%或PEG 0.8-1.2
- **高估**：历史PE分位数>70%或PEG>1.5

## 行业差异化关注

### 消费品行业
- **核心关注**：品牌力、渠道覆盖、复购率
- **关键指标**：毛利率、ROE、经营现金流
- **估值方法**：PE为主，结合PEG

### 科技行业
- **核心关注**：研发投入、技术迭代、产品生命周期
- **关键指标**：研发费用率、专利数、营收增速
- **估值方法**：PS为主，结合用户数、DAU等运营指标

### 制造业
- **核心关注**：成本控制、产能利用率、供应链管理
- **关键指标**：毛利率、存货周转、应收账款周转
- **估值方法**：PE、EV/EBITDA

### 金融行业
- **核心关注**：资产质量、资本充足率、净息差
- **关键指标**：ROE、不良贷款率、成本收入比
- **估值方法**：PB为主，PE为辅

### 地产行业
- **核心关注**：去化率、土储质量、杠杆水平
- **关键指标**：毛利率、净负债率、现金流
- **估值方法**：PE、NAV（净资产价值）法

## 财务造假识别

### 红旗信号
- **营收增长与现金流背离**：营收增长但经营现金流恶化
- **异常的高毛利率**：显著高于行业平均水平
- **频繁的商誉减值**：激进并购带来的风险
- **关联交易占比高**：可能存在利润操纵
- **频繁更换审计机构**：可能存在财务问题

### 验证方法
- 交叉验证：收入与纳税、用电量、运费等指标匹配
- 纵向对比：与历史数据对比，寻找异常波动
- 横向对比：与同行业公司对比
- 现金流验证：关注现金流与利润的匹配度
