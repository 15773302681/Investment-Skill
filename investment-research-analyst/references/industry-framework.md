## 概览
本框架提供系统性的行业研究方法论，涵盖行业生命周期、产业链结构、竞争格局、市场空间等核心维度，帮助建立对行业的深度理解。

## 行业生命周期分析

### 四阶段划分
- **导入期**：技术不成熟，市场教育阶段，盈利困难
  - 关键特征：规模小、增速快但波动大、竞争者少
  - 投资策略：高风险高收益，关注技术领先者
  - 案例：元宇宙、量子计算

- **成长期**：需求爆发，市场快速扩张
  - 关键特征：规模快速增长、竞争加剧、盈利能力分化
  - 投资策略：选龙头，关注市场份额提升
  - 案例：新能源汽车、光伏

- **成熟期**：需求稳定，格局固化
  - 关键特征：增速放缓、竞争格局稳定、现金流充沛
  - 投资策略：关注分红、估值修复、细分赛道机会
  - 案例：家电、白酒

- **衰退期**：需求萎缩，替代品出现
  - 关键特征：负增长、利润率下滑、产能过剩
  - 投资策略：避免配置或寻找转型机会
  - 案例：传统PC、胶卷

### 判断指标
- 行业增长率 vs GDP增速
- 渗透率水平
- 竞争者数量及市场份额
- 盈利能力变化趋势

## 产业链分析

### 分析框架
1. **产业链拆解**
   - 上游：原材料、核心零部件、供应商集中度
   - 中游：制造、加工、组装环节
   - 下游：应用场景、客户结构、渠道

2. **价值分布**
   - 各环节毛利率对比
   - 各环节利润占比
   - 价值链迁移趋势（微笑曲线）

3. **话语权分析**
   - 各环节议价能力
   - 进口依赖度
   - 技术壁垒

### 案例：新能源汽车产业链
- **上游**：锂矿、钴矿、镍矿 → 资源垄断，话语权强
- **中游**：正极材料、负极材料、电解液、隔膜 → 竞争激烈，技术迭代快
- **下游**：整车制造、充电桩 → 品牌与渠道竞争激烈

## 竞争格局分析

### 市场结构类型
- **完全垄断**：单一企业占100%市场份额
  - 案例：国家电网、中国烟草
  - 投资价值：确定性高，但价格受管制

- **寡头垄断**：少数几家头部企业主导
  - 案例：白酒（茅五泸）、快递（顺丰京东通达系）
  - 投资价值：格局稳定，盈利能力强

- **垄断竞争**：竞争者多，但各有差异化优势
  - 案例：家电、餐饮
  - 投资价值：选品牌力强的龙头

- **完全竞争**：无差异化，价格竞争激烈
  - 案例：大宗商品、低端制造
  - 投资价值：关注成本优势

### 分析维度
- **集中度指标**：CR3、CR5、CR10（前N大企业市场份额）
- **竞争维度**：价格、产品、渠道、品牌、技术
- **进入壁垒**：资本、技术、牌照、网络效应
- **退出壁垒**：资产专用性、沉没成本

### 竞争策略分类
- 成本领先：规模效应、供应链优化
- 差异化：品牌、技术、服务
- 聚焦：细分市场深耕

## 行业空间测算

### 测算方法

#### 方法1：TAM-SAM-SOM模型
- **TAM (Total Addressable Market)**：总潜在市场规模
- **SAM (Serviceable Addressable Market)**：可服务市场（产品能覆盖）
- **SOM (Serviceable Obtainable Market)**：可获得市场（实际能拿下）

示例：
- TAM：全球汽车市场（年销量8000万辆）
- SAM：中国汽车市场（年销量2500万辆）
- SOM：新能源汽车市场（年销量600万辆）

#### 方法2：渗透率模型
```
行业空间 = 基数 × 渗透率目标 × 客单价
```

案例：储能行业
- 基数：全球新增风光装机量
- 渗透率目标：配储比例从当前的10%提升至30%
- 客单价：单位储能成本

#### 方法3：量价拆分
```
行业空间 = 销量 × 单价
```

分析驱动因素：
- 销量增长：新用户、复购率、替代需求
- 单价变化：产品升级、通胀、竞争格局

### 数据来源
- 行业协会数据（中汽协、IDC、Gartner等）
- 龙头公司年报及财报
- 第三方研报（券商、咨询公司）
- 政府统计公报

## 关键成功因素（CSF）

### 不同行业的核心KPI

| 行业类型 | 关键成功因素 | 核心KPI |
|---------|-------------|---------|
| 消费品 | 品牌力、渠道覆盖 | 品牌溢价、铺货率、复购率 |
| 科技 | 技术迭代、产品创新 | 研发投入、专利数、产品更新频率 |
| 制造 | 成本控制、质量稳定 | 毛利率、良品率、产能利用率 |
| 服务业 | 用户规模、网络效应 | DAU/MAU、ARPU值、获客成本 |
| 资源 | 资源储备、成本优势 | 储量、开采成本、产能规模 |

### 分析框架
1. 识别行业价值链中最重要的环节
2. 找出该环节的竞争要素
3. 评估企业在关键要素上的竞争力

## 风险识别

### 系统性风险
- 宏观经济下行
- 政策监管变化
- 技术替代风险
- 地缘政治冲击

### 行业特定风险
- **周期性行业**：需求波动大，价格周期波动
- **高杠杆行业**：融资环境变化影响大
- **技术密集型**：技术迭代快，淘汰风险高
- **资源依赖型**：原材料价格波动

### 风险评估框架
- **发生概率**：高/中/低
- **影响程度**：严重/中等/轻微
- **可控性**：能否通过仓位管理、对冲等方式控制

## 行业分析报告模板

### 1. 行业概览
- 行业定义及分类
- 产业链图谱
- 市场规模及增长趋势

### 2. 生命周期判断
- 当前所处阶段及依据
- 与可比行业的对标

### 3. 竞争格局
- 市场集中度（CR3/CR5）
- 主要参与者对比
- 竞争格局演变趋势

### 4. 关键驱动因素
- 需求端驱动：政策、消费升级、技术进步
- 供给端驱动：产能释放、成本下降、技术突破

### 5. 行业空间测算
- 当前市场规模
- 未来3-5年市场空间预测
- 增长驱动因素

### 6. 投资逻辑
- 行业投资价值
- 推荐标的及理由
- 风险提示
