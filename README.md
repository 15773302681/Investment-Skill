# 投研经理工作流 Skill

## 项目简介

本 Skill 提供完整的投资研究工作流支持，涵盖从宏观经济分析到行业深度研究、研报获取、财务分析和投资报告撰写的全流程能力。专为投研经理、证券分析师和投资从业者设计。

## 核心功能

- **宏观经济分析**：经济周期研判、政策环境评估、外部影响分析
- **行业深度研究**：生命周期分析、产业链梳理、竞争格局评估、市场空间测算
- **研报自动获取**：支持东方财富等主流研报平台的数据爬取
- **财务指标分析**：盈利能力、成长性、财务健康度、估值建模
- **投资报告撰写**：标准化投研报告模板，覆盖完整分析维度

## 快速开始

### 环境要求

- Python 3.7+
- 网络连接（用于获取研报数据）

### 安装依赖

```bash
pip install -r requirements.txt
```

或手动安装：

```bash
pip install requests>=2.31.0
pip install beautifulsoup4>=4.12.0
pip install lxml>=4.9.0
```

### 使用方式

#### 1. 通过智能体调用（推荐）

加载 Skill 后，直接与智能体对话：
- "帮我分析新能源汽车行业"
- "获取半导体行业最近30天的研报"
- "分析某家公司的财务状况并撰写投资报告"

智能体会自动调用相应工具和框架完成分析。

#### 2. 直接调用脚本

获取行业研报：

```bash
python scripts/fetch_reports.py --industry "新能源汽车" --days 30
```

参数说明：
- `--industry`：行业名称（必填）
- `--days`：获取近N天的研报（可选，默认30天）
- `--source`：数据源选择（可选，默认 eastmoney，支持 sina/all）
- `--output`：输出文件路径（可选）

## 工作流程

### 标准投研流程

1. **确定研究范围**：明确研究目标（宏观/行业/公司层面）
2. **宏观环境分析**：基于 `references/macro-framework.md` 分析经济周期和政策环境
3. **行业研究**：调用脚本获取研报，基于 `references/industry-framework.md` 分析行业
4. **财务分析**：基于 `references/financial-metrics.md` 评估财务指标和估值
5. **报告撰写**：使用 `assets/report-template.md` 模板生成投研报告

### 快速研报获取

如仅需获取行业研报数据，可直接调用 `fetch_reports.py` 脚本。

## 典型应用场景

### 场景1：行业深度研究

**需求**：分析某行业的投资价值

**执行**：
1. 调用脚本获取该行业最新研报
2. 基于行业分析框架评估生命周期和竞争格局
3. 基于宏观框架分析政策影响
4. 生成行业研究报告

### 场景2：公司财务分析

**需求**：评估某上市公司的投资价值

**执行**：
1. 获取公司财务数据
2. 基于财务指标框架分析盈利能力、成长性、财务健康度
3. 进行多维度估值分析（PE、PB、DCF等）
4. 输出投资评级和目标价

### 场景3：宏观环境研判

**需求**：研判当前宏观环境对投资的影响

**执行**：
1. 基于宏观框架分析经济周期位置
2. 评估货币政策和财政政策立场
3. 分析外部环境（美联储、汇率、地缘政治）
4. 输出宏观投资建议

## 注意事项

1. **研报爬取限制**：受网站反爬机制影响，建议分批次获取，避免短时间内大量请求
2. **数据时效性**：研报数据有一定延迟，建议结合最新市场动态进行判断
3. **投资风险**：本工具提供分析支持，不构成投资建议，投资决策需结合自身风险承受能力
